# Ship's Logs

This folder contains issue reports and operational logs following the nautical theming of the Santiago architecture.

## Template

Use `ships-log-template.md` as the starting point for new issue reports and operational logs.

## Structure

- Log entries use markdown format with structured headers
- Include timestamps, severity levels, and KG integration metadata
- Follows chronological ordering for operational tracking

## Related

- Populated by santiago-pm when tracking issues and operational events
- Integrated with knowledge graph for incident management
