# Cargo Manifests

This folder contains feature specifications and requirements documents following the nautical theming of the Santiago architecture.

## Template

Use `cargo-manifest-template.md` as the starting point for new feature specifications.

## Structure

- Feature files use `.feature` extension with Gherkin syntax
- YAML frontmatter includes metadata for KG integration
- Follows scaffold pattern established by santiago-pm

## Related

- Populated by santiago-pm when creating new domain features
- Integrated with knowledge graph for requirements tracking
