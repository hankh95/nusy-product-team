# Crew Manifests

This folder contains agent role specifications and crew member profiles following the nautical theming of the Santiago architecture.

## Template

Use `crew-manifest-template.md` as the starting point for new agent role specifications and crew profiles.

## Structure

- Manifests define agent capabilities, responsibilities, and behavioral guidelines
- Include ethical frameworks and operational boundaries
- Integrated with knowledge graph for agent coordination

## Related

- Populated by santiago-pm when defining new agent roles
- Connected to strategic-charts for architectural context
