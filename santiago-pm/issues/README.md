# Issues Management

This folder contains issue tracking and management assets for Santiago development.

## Structure

- `README.md` - This file
- `issue-template.md` - Template for creating new issues
- `issue-workflow.md` - Issue lifecycle and workflow definitions
- `issue-types/` - Specific issue type definitions

## Integration

Issues are managed through the knowledge graph and connected to:

- Tasks and chores for implementation
- Code changes and commits
- Testing and validation
- Release management

## Types

- **Bug**: Software defects and errors
- **Feature**: New functionality requests
- **Enhancement**: Improvements to existing features
- **Technical Debt**: Code quality and maintenance issues
- **Documentation**: Documentation updates and improvements</content>
