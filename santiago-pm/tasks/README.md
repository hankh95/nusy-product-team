# Tasks Management

This folder contains task definitions and management assets for Santiago development.

## Structure

- `README.md` - This file
- `task-template.md` - Template for creating new tasks
- `task-workflow.md` - Task lifecycle and workflow definitions
- `task-types/` - Specific task type definitions

## Integration

Tasks are the implementation units that break down issues into actionable work items. They connect to:

- Issues: Parent issue that spawned the task
- Chores: Smaller maintenance activities
- Code changes: Actual implementation
- Testing: Validation of task completion

## Types

- **Development**: Code implementation and feature development
- **Testing**: Unit tests, integration tests, validation
- **Documentation**: Writing and updating documentation
- **Research**: Investigation and analysis tasks
- **Review**: Code review and quality assurance
- **Deployment**: Release and deployment activities
