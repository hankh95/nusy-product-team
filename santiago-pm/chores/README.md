# Chores Management

This folder contains chore definitions and management assets for Santiago development.

## Structure

- `README.md` - This file
- `chore-template.md` - Template for creating new chores
- `chore-workflow.md` - Chore lifecycle and workflow definitions
- `chore-types/` - Specific chore type definitions

## Integration

Chores are small, routine maintenance activities that support development. They connect to:

- Tasks: Supporting maintenance for larger tasks
- Issues: Small fixes and improvements
- Code quality: Linting, formatting, cleanup
- Infrastructure: Tool updates, dependency management

## Types

- **Code Quality**: Linting, formatting, cleanup
- **Dependencies**: Package updates, security patches
- **Infrastructure**: Tool configuration, CI/CD updates
- **Documentation**: Minor documentation updates
- **Testing**: Test maintenance, coverage improvements
- **Performance**: Optimization and monitoring tasks
